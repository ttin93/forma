Hand it to Claude Code

01

Spin up a new repo

Don't build inside this design project. Create a fresh git repo for the codebase — keep design and code separate.

02

Move CLAUDE.md to repo root

Copy the entire claude-code-spec/ folder into the new repo, then move CLAUDE.md up to the root. Claude Code reads it automatically on every session.

03

Copy the design canvas alongside

Put Forma — Full Application.html and its .jsx + tokens.css into a design/ folder next to the spec. Claude Code reads both formats.

04

First prompt — ground it before coding

# your first message to Claude Code
Read CLAUDE.md and claude-code-spec/BRIEF.md first. Then open
design/Forma — Full Application.html so you understand the visual
language. Don't start coding yet — give me a one-page summary of
what you understood and any blocking questions before we start
Phase 0 from claude-code-spec/ROADMAP.md.
05

Go phase by phase

Each phase in ROADMAP.md has acceptance criteria. After every phase, get a test report + a diff summary before moving on. Don't let it gallop ahead.

06

When the UI drifts — point at the canvas

"Open design/Forma — Full Application.html, look at the ConfigBuilder artboard. The inspector has tabs Field / Pricing / Validation / Logic. Yours is missing Logic — fix it." Claude Code can read your JSX exactly.

07

When you need new screens — come back here

If a flow turns out to need a screen the canvas doesn't cover (CSV import, error states, email templates), come back to this design tool. I'll add it to the canvas. You copy it into design/ in the build repo. Claude Code implements.

Without specs this thorough, the same build is 10–12 weeks because half of it is the engineer guessing what you meant and rebuilding. With Claude Code reading clear contracts, you're shaving real time.

One more thing

As you build, anything Claude Code keeps getting wrong — uses the wrong library, sneaks any into TypeScript, puts logic in components — open CLAUDE.md and add a line under "DON'T". It re-reads that file every session. After a week the file will be tuned to your codebase and Claude Code will be borderline psychic.
